Para importar el proyecto, desde eclipse hacer 
import ->existente maven project, seleccionas la carpeta Logging.

para ejecutar el proyecto -> Se tiene que ejecutar la clase Logs, que esta
dentro de ejecutable, es un formulario hecho con swing.

Para la conexion a bd, se tienen properties los cuales se pueden configurar
las credenciales estos son:
connectionProps.properties
credenciales.properties

Se recomienda tener sql instalado y crear una bd , el nombre de la tabla
se llama Log, los campos son mensaje tipo string, valor de tipo int. 

Se adjunta un backup de la bd en sql tambien.
